Learn How To Write Chinese

This website is an interactive resource for learning about simplified Chinese characters through their visual history and structure. Each character is presented with its evolution from ancient writing systems and stroke order.

Many Chinese characters were derived from visual representations of ideas. Understanding why each character looks the way it does helps with recognising and writing them. This resource focuses on nine common characters that also function as radicals.
